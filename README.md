# 🤖 AI Resume Intelligence System

An AI-powered Resume Screening and Candidate Ranking System built using Python, Streamlit, NLP, and Machine Learning techniques.

---

## 🌐 Live Demo

🔗 https://ai-resume-intelligence-hr.streamlit.app

## 📌 Overview

This application helps recruiters compare resumes against a Job Description and identify the most suitable candidates automatically.

The system extracts text from resumes, preprocesses the data, identifies relevant skills, calculates similarity scores, ranks candidates, and provides recommendations.

---

## ✨ Features

- 📄 Upload multiple PDF resumes
- 📝 Paste Job Description
- 🤖 AI-based Resume Analysis
- 📊 Resume Match Score
- 🏆 Candidate Ranking
- 📈 Analytics Dashboard
- 💡 Skill Recommendations
- 📂 Resume Parsing using PDFPlumber
